Framerate

JavaScript | PostgreSQL | React

A website for rating weapons and frames from the video game Warframe. Requires user registration and sign-in to submit ratings. User login information is hashed and stored using a PostgreSQL database along with the user's ratings for each item. Users can rate an item multiple times, overriding their previous rating instead of adding a new rating.

Warp Bound

JavaScript | Phaser 3

A bullet hell style game made using Phaser 3 framework. Unlike traditional bullet hell games, the only method of movement is to teleport around. Survive against an endless wave of enemies to obtain as much gold as possible. Featuring high score system and a single tap setting to teleport and attack with one click.
