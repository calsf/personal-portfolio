Framerate

An interactive website made using React.js for rating weapons and frames from the video game Warframe. Requires user registration and sign-in in order to rate. User login information is encrypted and stored using a PostgreSQL database along with the user's ratings for each item. Users can rate an item multiple times, overriding any previous rating rather than adding a new rating.

Home Page

A website meant to serve as a suitable home page. Features include weather information using OpenWeatherMap API integration, Google search bar, quick links, notepad, and customizable settings. No user registration is required as settings are saved using local storage.

Mini Arcade

A first time experience at creating a small 2D side-scrolling/platform game using Unity 2017.1.1. The game consists of two different game modes with three levels each along with their own controls and gameplay mechanics. Graphical assets are made in Adobe Photoshop. No sound is present in the game.

Leap Frog

A more fleshed out 2D side-scrolling/platform game building upon experiences from the Mini Arcade project. Features currently include level selection, variety of collectable weapons, saving of level completion and weapon acquisition, pausing, and customizable keybinds. Graphical assets are made in Adobe Photoshop. Sound will be added.
