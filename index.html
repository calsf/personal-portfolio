Framerate

An interactive website made using React.js for rating weapons and frames from the video game Warframe. Requires user registration and sign-in in order to rate. User login information is hashed and stored using a PostgreSQL database along with the user's ratings for each item. Users can rate an item multiple times, overriding any previous rating rather than adding a new rating.

Warp Bound

A bullet hell style game made using Phaser 3 framework and Webpack module bundler. Unlike traditional bullet hell games, the only method of movement is to teleport around. Survive against an endless wave of enemies to obtain as much gold as possible. Featuring high score system and a single tap setting to teleport and attack with one click.

Leap Frog Jones

A 2D side-scroller/platformer game made using Unity 2018.3.7. Play as a frog named Jones as he ventures through his home forest. Featuring 5 levels along with level selection, a variety of collectable weapons, automatic saving of level completion and weapon acquisition, high score system, and customizable keybinds. Graphical assets are made in Adobe Photoshop.

A discord bot for the video game Warframe made using discord.py and WarframeStatus API. Bot includes commands for obtaining Warframe world state information as well as tracking certain world state information for each user.
